# Mise en production projet d'apprentissage

## Réalisation
Dans le cadre de notre projet d'apprentissage, nous avons réalisé une application flask.
Nous n'avons pas utilisé de server spring puisque l'application est très simple et ne nécessite pas de fonctionnalités avancées.
Une application spring est une application lourde et lance un server tomcat, ce qui n'est pas nécessaire pour une one page application.
Il est néanmoins possible de plug l'application flask sur un autre front si besoin puisque l'application flask peut facilement être transformée en API REST.


## Prérequis
- Installer Docker
- Installer Docker-compose
- Installer Git

## Installation
- Cloner le projet
- Se placer dans le dossier du projet
- Lancer la commande `docker-compose up `

## Utilisation
Une fois le projet lancé, se rendre sur l'adresse `localhost:5000` pour accéder à l'application
